   ### URL Shortener Microservice   

    
Example usage:

   - https://scandalous-koala.glitch.me//new/http://github.com
   
   - https://scandalous-koala.glitch.me/27
   
Example output:
   
   ```js
        {
            original_url: "https://www.google.com",
            short_url: "https://scandalous-koala.glitch.me/27"
        }
   ```
